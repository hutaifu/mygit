# 预编译过程，预编译发生在函数执行的前一刻，
### 函数里面预编译
1.创建AO对象 Activation Object (执行期上下文)  可以理解为作用域
2. 找函数的形参和变量声明，将变量和形参名作为AO属性名，值为undefined.
3.将实参和形参相统一
4.在函数体里面找函数声明，值赋予函数体(注意，最新版本if中声明的函数不起效)

## 注意函数表达式不能算作函数声明。
