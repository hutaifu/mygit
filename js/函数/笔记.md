# 实参列表
函数自带隐式参数arguments,实参组成的数组，

# 形参长度，函数名.length。
function sum(){
    sum.length;
}

实参数组和形参存在映射规则，一个变，另一个也跟着变。
如果实参列表个数和形参没有对应 ，没有对应的形参不存在映射规则；
